== Description ==
Do you want to change the color of your site navigation bar?
Tired of the simple, default style of displaying navigation bar colors on your site?
The "magic scroller" plugin is the best option for you.
This plugin is the lightest and easiest plugin to change the color and style of scroll bar.
* Currently the language of this plugin is Persian *
Important features of the plugin:
* Change the color of the scroll bar
* Change the background color of the scroll bar
* Change the color of the hover scroll bar
* Change the width of the load scroll
* Rounding the scroll bar
* Has english settings panel
* Has a complete user guide
== Installation ==
To install the plugin, enter the plugins section in the WordPress counter and search for the plugin name and install the plugin.
Then a section called "Change the color of the scroll bar" will appear for you in the WordPress counter
== Frequently Asked Questions ==
